This guide is made for those with a fast paced life, looking for a job in web development.This beginner's guide is focused on
approachable projects to help you gain the skills needed to become a web developer. Each section contains a brief summary
of the topic, a list of external resources i have used and found the most helpful, a small cheat sheet, and project prompts.
